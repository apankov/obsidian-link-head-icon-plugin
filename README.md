## How to install

1. Go to **Community Plugins** in your [Obsidian](https://www.obsidian.md) Settings and **disable** Safe Mode
2. Click on **Browse** and search for „Link Head“
3. Click install
4. Toggle the Plugin on in the **Community Plugins** Tab
